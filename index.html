Projects

Deep Neural Network Based Surrogate Model for Earthquake Rupture Simulation

Supervisor: Associate Professor Chao Liang, IDMR, Sichuan University

Duration: Feb. 2023 - Apr. 2024

In this project, I contributed to the generation of a large dataset by running over 300,000 simulations, providing insights into the rupture dynamics process, including phenomena like supershear transition. The project focuses on developing a deep neural network-based surrogate model to accelerate earthquake dynamic rupture (EDR) simulations.

Boundary Element Method for Fluid-filled Crack

Supervisor: Associate Professor Chao Liang, IDMR, Sichuan University

Duration: May 2023 - Present

This ongoing project involves reproducing existing work (Pointer et al., 1998) on fluid-filled cracks and further developing the Boundary Element Method (BEM) to solve multi-scale wave problems. The aim is to infer crack geometry and fluid properties from seismic data and simulations.

Quadlayer Krauklis Wave Model for Explaining VLP Tremors in the Gulf of Guinea

Supervisor: Associate Professor Chao Liang, IDMR, Sichuan University

Duration: Apr. 2024 - Present

This project focuses on deriving a rigorous model for VLP tremors using a quad-layer system, which includes an inviscid ocean, a thin crust, a viscous magmatic sill, and elastic half-space. The project aims to provide a more accurate understanding of VLP tremor excitation mechanisms.
